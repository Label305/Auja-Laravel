# Auja-Laravel

Auja for Laravel.

## Setup

To start developing for this project, clone [Auja-Laravel-Example](https://github.com/Label305/Auja-Laravel-Example).
After running `composer install`, you can find this project in `\vendor\label305\auja-laravel\`.
